About

Help you to reload the dropdown list options once target dropdown list is changed.

Requirement Latest version of jQuery 

<script type='text/javascript' src='url_to/jquery.min.js' />

Installation

<script type='text/javascript' src='url_to/jquery.dropdown_list_chain.js' />

Usage

<script type='text/javascript'>
  jQuery(function($) {
    $(chainer_selector).chain( /* { chainee_selector | jQuery | Element } [ , options = {} ] */ );
    $(chainee_selector).chainedTo( /* { chainer_selector | jQuery | Element } [ , options = {} ] */ );
  });
</script>
NOTE: the chainer doesn't need to be a dropdown list, it could be input field, checkbox, radiobox or even a textarea. Please see Local -> Option 2 -> Demo.
NOTE: the chainee must be HTML Select element.

Options

Precedence:

  1. Options passed in javascript.

    e.g. $('selector').chain('selector', { include_blank: false });

  2. Options set in the HTML

    e.g. <select data-include_blank='false' />

  3. Default options
Name Type Default Description
include_blank object
{
    text: ' - ',
    value: ''
}

Whether to include the blank option for dropdown list as the first item.

You could disable this by setting it to false

include_blank.text string ' - ' Text for the blank option.
include_blank.value string '' Value for the blank option.
remember_last_value boolean true Whether to remember the last selected value
ajax object false

1. Options for jQuery AJAX call, see jQuery.ajax( settings )

2. or you coulf pass a function to return options for jQuery AJAX.

param: $chainer , $chainee

return: object

For example:

function($chainer, $chainee) {
 return {};
}
ajax_mapping object
{
    text: ' - ',
    value: '',
    build_option: false
}
 Options for setting up how to map each AJAX record and build the HTML Option element.
ajax_mapping.filter function false

Custom method to filter each AJAX record.

param: record , chainer_value

return: boolean

Example: 

function(record, chainer_value) {
 return chainer_value == record;
}
ajax_mapping.text string 'text' Property name of each AJAX record which will be used as the text
ajax_mapping.value string '' Property name of each AJAX record which will be used as the value
ajax_mapping.build_option function false

Custom method to map each AJAX record to build the HTML Option element.

param: record

return: option ( jQuery | Element )

Example: 

function(record) {
 return $('<option />').text(record.text).attr('value', record.value)
}

Example - Local

Option 1

No javascript is needed if you follow below markup.

<!-- Chainee -->
<select id='...' data-target='#{ parent_css_selector }' data-toggle='chain'>
  <option value='...' data-chain='...'> ... </option>
</select>

Code Example

<!-- Chainer -->
<select id='country'>
  <option value='au'> Australia </option>
  <option value='cn'> China </option>
</select>

<!-- Chainee -->
<select id='city' data-target='#country' data-toggle='chain'>
  <option value='syd' data-chain='au'> Sydney </option>
  <option value='mel' data-chain='au'> Melbourne </option>
  <option value='bj' data-chain='cn'> Bei Jing </option>
  <option value='gz' data-chain='cn'> Guang Zhou </option>
</select>

Demo

Option 2

Manual Javascript setup.

Code Example

<!-- Chainer -->
<input id='species' />

<!-- Chainee -->
<select id='breed'>
  <option value='' data-chain='dog'> Sydney </option>
  <option value='mel' data-chain='dog'> Melbourne </option>
  <option value='bj' data-chain='cat'> Bei Jing </option>
  <option value='gz' data-chain='cat'> Guang Zhou </option>
</select>

<script type='text/javascript'>
  jQuery(function($) {
    $('#country').chain('#city');
    // or $('#city').chainedTo('#country');
  });
</script>

Example - Remote

See jQuery.ajax( settings ) for more information about the jQuery AJAX call.

Code Example

<!-- Chainer -->
<select id='gender' />
  <option />
  <option value='male'> Male </option>
  <option value='female'> Female </option>
  <option value='unknown'> Unknown </option>
</select>

<!-- Chainee -->
<select id='person_name'></select>

<script type='text/javascript'>
  jQuery(function($) {
    $('#gender').chain('#person_name', {
      ajax: {
        url: './examples/demo.json',
        dataType: 'json'
      },
      ajax_mapping: {
        filter: function(record, chainer_value) {
          if (record.chain !== chainer_value) return true;
        }
      }
    });
  });
</script>
