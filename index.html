Usage

<script type='text/javascript'>
  jQuery(function($) {
    $(chainer_selector).chain( /* { chainee_selector | jQuery | Element } [ , options = {} ] */ );
    $(chainee_selector).chainedTo( /* { chainer_selector | jQuery | Element } [ , options = {} ] */ );
  });
</script>
NOTE: the chainer doesn't need to be a dropdown list, it could be input field, checkbox, radiobox or even a textarea. Please see Local -> Option 2 -> Demo.
NOTE: the chainee must be HTML Select element.

Local

Option 1

No javascript is needed if you follow below markup.

<!-- Chainee -->
<select id='...' data-target='#{ parent_css_selector }' data-toggle='chain'>
  <option value='...' data-chain='...'> ... </option>
</select>

Code Example

<!-- Chainer -->
<select id='country'>
  <option value='au'> Australia </option>
  <option value='cn'> China </option>
</select>

<!-- Chainee -->
<select id='city' data-target='#country' data-toggle='chain'>
  <option value='syd' data-chain='au'> Sydney </option>
  <option value='mel' data-chain='au'> Melbourne </option>
  <option value='bj' data-chain='cn'> Bei Jing </option>
  <option value='gz' data-chain='cn'> Guang Zhou </option>
</select>

Option 2

Manual Javascript setup.

<!-- Chainer -->
<input id='species' />

<!-- Chainee -->
<select id='breed'>
  <option value='' data-chain='dog'> Sydney </option>
  <option value='mel' data-chain='dog'> Melbourne </option>
  <option value='bj' data-chain='cat'> Bei Jing </option>
  <option value='gz' data-chain='cat'> Guang Zhou </option>
</select>

<script type='text/javascript'>
  jQuery(function($) {
    $('#country').chain('#city');
    // or $('#city').chainedTo($('#country'));
  });
</script>
