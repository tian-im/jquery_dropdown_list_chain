Usage

Local

Option 1

No javascript is needed if you follow below markup.

<select id='...' data-target='#{ parent_css_selector }' data-toggle='chain'>
  <option value='...' data-chain='...'> ... </option>
</select>
Code Example 
<select id='country'>
  <option value='au'> Australia </option>
  <option value='cn'> China </option>
</select>

<select id='city' data-target='#country' data-toggle='chain'>
  <option value='syd' data-chain='au'> Sydney </option>
  <option value='mel' data-chain='au'> Melbourne </option>
  <option value='bj' data-chain='cn'> Bei Jing </option>
  <option value='gz' data-chain='cn'> Guang Zhou </option>
</select>
