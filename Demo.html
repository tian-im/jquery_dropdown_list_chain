Example

Requirement

  1. <script type='text/javascript' src='path_to/jquery.min.js' />

Installation

  1. <script type='text/javascript' src='path_to/jquery.in_field_label.js' />

Usage

  1. <input type='text' class='in_field_label' />

OR

  1. <script type='text/javascript'>
  2.   jQuery(function($){
  3.     $('input').in_field_label()
  4.   })
  5. </script>

Options

Key Description
align Values: left right
padding Padding will be used with align to position the label. Number ONLY.
opacity Opacity for the label when the input field is focused but has no value.
opacity_min Min opacity for the label when the input field is focused but has no value.
opacity_max Max opacity for the label when the input field is focused but has no value.
tag_class For additional styling.
